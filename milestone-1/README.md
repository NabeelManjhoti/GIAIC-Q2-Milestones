A simple introductory project to get started with Next.js by setting up the environment and creating a "Hello World" page. This project is focused on building familiarity with Next.js and configuring it from scratch.

Milestone 1: Installation and "Hello World" in Next.js
Overview
This project is a step-by-step guide for setting up a Next.js application, from installation to creating a basic "Hello World" page. The milestone aims to establish a working environment and test the initial configuration by displaying a basic greeting on the main page.
